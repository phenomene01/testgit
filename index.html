Pour créer un depot dans le terminal

Allez dans le dossier que vous souhaiter mettre sur github

Exemple : cd /var/www/html/testgit/

initialiser le depot

taper la commande suivante: git init

Ajouter les modifications/nouveaux fichier

La ou vos nouveaux fichier ou modification sont : git add *

valider les modifications/ajouts

vous devez valider avec un commit : git commit

Maintenant vous devez créer un nouveau depot sur le site github

Une fois le dépot créer vous trouverez la section "an existing folder"

Executez dans votre terminal les commandes afficher dans cette section

Exemple:git remote add origin https://github.com/phenomene01/testgit.git

si votre chemin au dépot distant a bien été ajouté vous pourrez envoyer les modifications en ligne avec la commande suivante

git push -u origin master

Si vous voyez apparaitre les modifications sur le site github
